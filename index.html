Time Block Scheduling

Users will be able to input their classes or work schedules and create a proper time block to manage their weeks. Having this will allow the users to manage all their daily work around their classes or other activities.

Calendar Planning

Having a calendar implemented in our site will let users manage all their events. Using the inputs they put in from the time block scheduling, users can make sure they have everything down to date and time to make sure everything gets done.

Groups

One more feature we want to add is the use of groups. Having users create groups based around their workplace or school class will allow coworkers to schedule work or events properly. Having a chat function as well will greatly improve productivity with their collaboration.
